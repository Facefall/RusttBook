## 我们的承诺

Rustt 郑重承诺：**永不停摆，永远更新**。大家可以思考下，我们为何不用网站的形式而是用 Github 仓库的形式？原因之一就在于网站太容易倒闭了，而 Github 组织仓库不会，退一万步讲，就算未来有部分贡献者不再参与了，也不会影响组织的正常运行。
